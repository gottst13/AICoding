### 3.1 统一计费流程

```
1. 车辆入场 → 创建订单
   ↓
2. 系统自动记录进入初始区域
   ↓
3. 车辆出场 → 计算费用
   ↓
4. 应用免费时长、封顶价规则
   ↓
5. 生成订单账单
```

**关键代码**:
```java
// 统一计费策略
FeeResult result = unifiedFeeStrategy.calculate(context);
// 包含：免费时长检查、封顶价格应用、续时计费
```

---

### 3.2 分区计费流程（跨区停车）

```
1. 车辆入场 → 进入 A 区
   ↓
2. 系统创建第 1 段记录
   ↓
3. 车辆移动到 B 区
   ↓
4. 结束第 1 段，创建第 2 段
   ↓
5. 车辆出场
   ↓
6. 各段费用累加
```

**关键代码**:
```java
// 跨区分段计费
segmentService.exitZone(orderNo, zoneA, exitTime);
segmentService.enterZone(orderNo, zoneB, enterTime);
// 自动生成分段记录，独立计费
```
